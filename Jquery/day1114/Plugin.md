# Plugin
- 자주 사용되는 디자인이나 기능을 미리 구현해놓고 배포하는 것
- jquery.com에는 다양한 plugin 제공

사용법)
1. 검색
2. 기능분석
3. 사용)-외부파일방식
``<script type="text/javascript" src=""></script>``

## Summernote
- `<textarea>`나 `<div>`를 사용하여 사용자가 글자 이외에 다양한 대상을 입력할 수 있도록 고안된 JavaScript Library
- 이미지를 첨부하면 binary에서 text로 encoding되어 전송한다.
- back-end로 값을 전송할 때 반드시 POST방식으로만 전송이 가능하다.
