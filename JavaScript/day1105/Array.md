# Array
- 일괄처리 시 사용
- 일차원 배열만 지원
- 가변길이 형
- 배열에 방의 개수를 얻기 위해서 length 속성을 사용(배열명.length)
- 배열에 방에는 모든 데이터형을 입력할 수 있다(이전 방의 데이터 형과 다음 방의 데이터 형은 다를 수 있다-사용상 주의)

사용법)
1. 생성)<br>
`var 배열명=new Array();`<br>

	1. 초기화(기본형 형식의 사용)<br>
	`var 배열명=[값,,,];`

2. 값 할당) 
`배열명[인덱스]=값;`

3. 값 사용)
`배열명[인덱스];`

## 배열의 기능
- 가장 마지막 방 뒤에 값 추가(인덱스를 사용하지 않아도 된다.)
- push 추가할 값

- 방에 값이 존재하는지 검색
`indexOf(찾을 값)`
`var ind=배열명.indexOf(찾을 값);` 
방의 인덱스 값이 없다면 -1

- 배열방의 값을 삭제 (자르기): 배열 방의 크기가 줄어든다.
`splice(시작인덱스, 삭제할 방의 수)`
`배열명.splice(시작인덱스, 삭제할 방의 수)`

## 이차원 배열(가변배열)
- 이차원 배열은 제공하지 않으나, 일차원 배열을 사용하면 이차원 배열을 만들 수 있다.
- 행과 열로 구성된 배열
- 이차원 배열의 행은 일차원 배열로 구성되어 있다.

사용법)
1. 일차원 배열을 생성
`var 배열명=new Array();`

2. 일차원 배열의 방에 일차원 배열을 생성 <- 이차원 배열
`배열명[방의번호]=new Array();` <- 배열의 방의 값을 모를 때
`배열명[방의번호]=[값,,,];` <- 방의 값을 알 때

3. 값 할당
`배열명[행의 번호][열의 번호]= 값;`

4. 값 사용
`배열명[행의 번호][열의 번호]`

5. 일괄처리
```JavaScript
for(var i=0; i<배열명.length i++){
	for(var j=0; j<배열명[i].length; j++){
		배열명[i][j];
	}
}
```
