## Operator
- 연산에 사용되는 예약된 부호들
- 산수연산자, 관계연산자, 논리연산자, 문자열연산자, null 비교연산자, 범위 비교연산자, 문자열 붙임연산자 제공
- null은 연산되면 연산결과가 null이 나온다.
- 문자열 붙임연산자: 조회되는 칼럼에 사용할 수 있는 연산자

사용법
``칼럼명||'붙일문자열', '붙일문자열'||칼럼명``

### 산술연산자
- 조회되는 칼럼에서 사용될 수 있다.
- +, -, *, /, mod 함수를 사용하면 나눈 나머지를 구할 수 있다.
- where 절에서 사용하는 연산자

### 관계연산자
- 조회되는 칼럼에서 사용할 수 없다.
- >, <. >=, <=, =, !=
```
select
from
where 관계연산자
```

### 논리연산자
- 조회되는 칼럼에 사용할 수 없다.
- and, or, not
